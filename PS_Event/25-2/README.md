# 강의 배정

수뭉이는 새학기를 맞아 수강 신청을 해야한다.
수뭉이는 학교를 너무 오기 싫은 나머지 모든 수업을 하루에 몰아듣기로 결정했다.

수업은 3개의 값으로 표현 할 수 있다.
시작 시간 $s$, 종료 시간 $e$, 학점 $c$

수뭉이는 장학금을 받기 위해서 최대한의 학점을 얻을 수 있는 시간표를 짜려고 한다.
각 수업을 겹치지 않게 들을 때 얻을 수 있는 학점의 최댓값을 구해보자.
단, 앞의 수업의 종료 시간과 뒤의 수업의 시작 시간은 겹칠 수 있다.


## 입력

첫 번째 줄에 테스트케이스의 수 $T(1\leq T\leq 10)$가 주어진다.

각 테스트케이스의 입력은 다음과 같이 주어진다.
첫 번째 줄에는 수업의 수 $N$이 주어진다.

두 번째 줄부터 $N + 1$번째 줄에는 
각 수업의 시작 시간 $s_i$, 종료 시간 $e_i$, 학점 $c_i$가 주어진다.

## 서브태스크

| 번호 | 배점 | 제한 |
| --- | --- | --- |
| 1 | 10 | $1\leq N\leq 1,000, c_i = 1, e_i - s_i = 1, 1\leq s_i < e_i\leq 1,000$ |
| 2 | 20 | $1\leq N\leq 50,000, c_i = 1, 1\leq s_i < e_i\leq 50,000$ |
| 3 | 30 | $1\leq N\leq 1,000, 1\leq s_i < e_i\leq 1,000, 1\leq c_i\leq 1,000$ |
| 4 | 40 | $1\leq N\leq 50,000, 1\leq s_i < e_i\leq 50,000, 1\leq c_i\leq 1,000$ |

## 출력

얻을 수 있는 학점의 최댓값을 출력한다.
